Download
========

Latest Release
--------------

**Release 1.0** : Nov 21st. 2005

Download `ez_setup.py </files/ez_setup.py>`_ then run::
    
    python ez_setup.py Routes
    
Or if you already have Easy Install::
    
    easy_install Routes
    
Development Version
-------------------

You can checkout the latest Routes code from the svn repository with::
    
    svn co http://routes.groovie.org/svn/trunk routes
